Clicker service
===========================

First instal
----------------------------
1. Install MySQL
2. Install PHP
3. Install Node.js and NPM
4. Add mysql dump to db

Required node.js modules
-----------------------------
> cd node
> npm install

Starting node server
--------------------------
server.sh start [dev]

Adding the database
-----------------------------
CREATE DATABASE clicker CHARACTER SET utf8 COLLATE utf8_bin;
